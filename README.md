# Mini Radar + Semantic Company Search

Buscador semántico de empresas impulsado por inteligencia artificial, desarrollado para identificar compañías relevantes mediante consultas en lenguaje natural. El sistema combina procesamiento de datos, embeddings semánticos, scoring estratégico y una interfaz web moderna para explorar resultados de forma eficiente.

---

## Funcionalidades principales

- Búsqueda semántica de empresas mediante lenguaje natural.
- Generación de embeddings con modelos NLP.
- Ranking de resultados por similitud.
- Radar Score estratégico para clasificación empresarial.
- Etiquetado automático de empresas:
  - CANDIDATE
  - REVIEW
  - DISCARD
- Interfaz web desarrollada con Flask.
- Diseño moderno y orientado a producto real.

---

## Tecnologías utilizadas

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Sentence Transformers
- PyTorch
- HTML / CSS
